# Personal Portfolio Website

A full-stack portfolio website designed to showcase my projects, technical skills, professional experience, and academic background.

The website serves as a centralized platform where recruiters, hiring managers, and collaborators can explore my work across Data Science, Machine Learning, Analytics, Software Development, and Artificial Intelligence.

## Live Demo

**Website:** https://rita-sharipova-portfolio.up.railway.app/


## Overview

This portfolio was built to provide an interactive and professional overview of my technical projects, experiences, and accomplishments.

In addition to highlighting completed projects, the platform allows visitors to learn about my educational background, technical skills, and professional experiences while providing multiple ways to connect with me directly.


## Features

### Professional Project Showcase

* Detailed project descriptions
* Technology stacks for each project
* Direct links to GitHub repositories
* Live project demonstrations when available

### Experience & Leadership

The Experience section highlights:

* Professional work experience
* Academic involvement

* Direct contact information
